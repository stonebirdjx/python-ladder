# 逻辑值False

- 被定义为假值的常量: `None` 和 `False`。
- 任何数值类型的零: `0`, `0.0`, `0j`, `Decimal(0)`, `Fraction(0, 1)`
- 空的序列和多项集: `''`, `()`, `[]`, `{}`, `set()`, `range(0)`

# 按位运算

|          |                        |        |
| :------- | :--------------------- | :----- |
| 运算     | 结果：                 | 备注   |
| `x | y`  | *x* 和 *y* 按位 *或*   | (4)    |
| `x ^ y`  | *x* 和 *y* 按位 *异或* | (4)    |
| `x & y`  | *x* 和 *y* 按位 *与*   | (4)    |
| `x << n` | *x* 左移 *n* 位        | (1)(2) |
| `x >> n` | *x* 右移 *n* 位        | (1)(3) |
| `~x`     | *x* 逐位取反           |        |
